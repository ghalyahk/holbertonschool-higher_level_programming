# Holberton School - Higher Level Programming

This repository contains Python projects for the **Python Data Structures** tasks at Holberton School.

## Project: Print a List of Integers

This project includes a Python function that prints all integers of a list, one per line, using `str.format()`. No modules are imported and no type casting is used.

### Files

- `0-print_list_integer.py` — contains the function `print_list_integer(my_list=[])`.
- `0-main.py` — test file for the function.
