Python - Hello, World

This project is an introduction to Python programming.
It covers basic concepts such as running Python scripts, using print statements, working with strings, and understanding the Python interpreter.

Learning Objectives

By the end of this project, you should be able to explain to anyone, without the help of Google:

Why Python programming is awesome

How to run Python scripts

How to use the Python interpreter

How to print text and variables using print

What strings are and how to manipulate them

What indexing and slicing mean

How to check your code with pycodestyle
