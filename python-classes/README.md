# Python - Classes and Objects

This repository contains my solutions for the Holberton School project **"Python - Classes"**.

## Description
In this project, we learn:
- How to create classes in Python
- What is an object
- Attributes and methods
- The difference between public, protected, and private attributes
- How to define constructors (`__init__`)
- How to write docstrings

## Files
- `python-classes/0-square.py`: Defines an empty class `Square`.
- `python-classes/0-main.py`: Test file to run the first task.

## Requirements
- Python 3.10
- Ubuntu 20.04 LTS
- No external modules allowed unless specified
